# YUV_Raw_Data_Solvent
.yuv(raw data) to meaningful data

## Features
- adjustable width, height and yuv format (4-4-4) (4-2-2) (4-2-0)
- windows image selection pop-up (only .yuv)
- select pathname for results (bmp format)
